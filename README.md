# Decentralized Voting DApp

This project is a blockchain-based voting system built using Ethereum smart contracts.

## Features

• Admin can add candidates  
• Users connect wallet using MetaMask  
• Each wallet can vote only once  
• Votes stored permanently on blockchain  
• Transparent and tamper-proof voting  
• Live vote count
